Introduction to Python

This study guide provides a quick-reference summary of what you learned in this lesson and serves as a guide for the upcoming practice quiz.

Overview of Python

In this segment, you learned that Python is a general-purpose programming language commonly used for scripting, automation, and developing a variety of applications. Python is compatible with most operating systems, including Windows, Linux, and Mac OS, and is updated every few years. It can run on various machines, such as servers, workstations, PCs, mobile devices, IoT, and more.

Applications of Python

Python is widely used in the IT field, including:

  • IT support
  • System administration
  • Web development
  • Machine learning
  • Data analytics
  • And more!

Python can perform tasks such as calculating statistics, running e-commerce sites, processing images, and interacting with web services. Its syntax resembles the English language, making it easier to learn compared to other programming languages.

Key Characteristics of Python

Python is:

  • A general-purpose scripting language
  • A popular language for coding various applications
  • A frequently used tool for automation
  • A cross-platform compatible language
  • A beginner-friendly language

Python is not:

  • A platform-specific / OS-specific scripting language
  • A client-side scripting language
  • A purely object-oriented programming language

Code Comparison: Python vs. Bash

The following code illustrates a syntax difference between Python and Bash:

Print to screen in Python

print("Hello, how are you?")

Output: Hello, how are you?

Print to screen in Bash

echo Hello, how are you?

Output: Hello, how are you?

Key Terms

  • Platform-specific / OS-specific scripting language: Languages used by system administrators on specific platforms.
  • Client-side scripting language: Languages primarily used for web programming, executed in the user's browser.
  • Machine language: The lowest-level computer language, directly communicating in binary code.
  • Cross-platform language: A language compatible with multiple operating systems.
  • Object-oriented programming language: Languages where coding elements are considered objects with configurable properties.
  • Python interpreter: The program that reads and executes Python code.
